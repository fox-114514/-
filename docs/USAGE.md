# CloudAsset 使用手册

## 1. 登录

打开 Web 客户端后填写：

- 服务器地址：本地开发可留空；远程部署填写 `https://asset.example.com`
- API Key：服务端 `.env` 中的 `API_KEY`

登录信息会保存在浏览器本地存储中。更换服务器或 Key 时，点击右上角“登出”后重新登录。

## 2. 上传资产

进入“上传”页面：

1. 拖拽文件到上传区域，或点击选择文件。
2. 可填写显示名称、描述、标签。
3. 标签用英文逗号分隔，例如 `logo, v1, product`。
4. 点击上传，完成后会跳转到资产详情页。

服务端会自动记录：

- 原始文件名
- MIME 类型
- 文件大小
- SHA-256 哈希
- 上传时间
- 存储路径

## 3. 搜索和筛选

资产库首页支持：

- 按名称、原始文件名、描述搜索
- 按分类筛选：`image`、`video`、`audio`、`document`、`code`、`data`、`other`
- 按标签筛选

每页最多返回 100 条，Web 默认显示 50 条。

## 4. 预览和下载

详情页支持：

- 图片预览
- 视频播放
- 音频播放
- 文本、代码、JSON、CSV 等文本预览
- 其他文件下载查看

私有资产内容接口需要 API Key，Web 会先通过带鉴权的 `fetch` 拉取为 blob URL，再交给浏览器预览。

## 5. 编辑元数据

在资产详情页点击“编辑”，可以修改：

- 显示名称
- 描述
- 分类
- 标签

保存后只更新元数据，不会改动原始文件。

## 6. 删除资产

点击“删除”会同时删除：

- SQLite 中的资产记录
- 资产文件
- 标签关联

删除不可恢复，建议生产环境定期备份。

## 7. 公开分享

在详情页点击“生成分享链接”。分享链接格式：

```text
https://asset.example.com/#/s/<share_token>
```

分享页无需 API Key，可以查看公开元数据、预览支持的文件类型、下载原文件。

取消分享后，原链接立即失效。

## 8. Android 客户端

安装 APK 后：

1. 输入服务器地址，例如 `https://asset.example.com`。
2. 输入 API Key。
3. 连接成功后可以搜索、预览图片、上传文件、生成分享链接。

Android 当前是轻量客户端，复杂预览建议使用 Web 端。

## 9. MCP Agent 使用

Agent 接入后可以执行：

- 列出资产
- 搜索资产
- 读取文本资产
- 上传新资产
- 更新标签和描述
- 删除资产
- 生成分享链接

远程 HTTP MCP 必须传入 `X-API-Key` 或 `Authorization: Bearer <API_KEY>`。

## 10. 备份建议

Docker 部署默认数据在两个 volume：

- `cloudasset-data`：SQLite 数据库
- `cloudasset-files`：上传文件

建议至少每周备份一次，重要资产库建议每天备份。
